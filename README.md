# 🛡️ TrustLens: National Threat Intelligence & Verification Platform

**TrustLens** is a decentralized, AI-powered command center designed to detect, analyze, and neutralize misinformation and cyber threats in real-time. Unlike traditional fact-checkers, TrustLens operates as a **Cyber-Physical Threat Intelligence System**, combining Agentic AI with immutable blockchain audits.

---

## 🚀 Key Capabilities

### 🧠 1. Agentic AI Core (Gemini 2.5)
* **Contextual Understanding:** Goes beyond keyword matching to understand *intent* (e.g., detecting urgency in phishing emails).
* **Multi-Modal Verification:** Cross-references claims against a live knowledge base of trusted global sources (BBC, ISRO, NASA).

### ⛓️ 2. Immutable Ledger (Blockchain)
* **Tamper-Proof History:** Every verified news item is hashed (SHA-256) and anchored to a local blockchain.
* **Audit Trail:** Users can retrieve original abstracts using the Hash ID to prove data integrity years later.

### 🛡️ 3. Real-Time Threat Map & Heuristics
* **Cyber Shield:** Instantly blocks known phishing patterns (`free-iphone`, `urgent-kyc`) using a deterministic rule engine.
* **Smart Geolocation:** Traces server origins to detect anomalies (e.g., an Indian local newspaper hosted on a suspicious Russian server).

### 📊 4. Cognitive Bias Radar
* **Sentiment Analysis:** Quantifies "Emotional Intensity" and "Subjective Bias" to flag sensationalist propaganda.
* **Visual Confidence:** Dynamic confidence bars updated in real-time based on a weighted trust formula.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Cyberpunk/Defense UI), JavaScript (Live Ticker, Dynamic Charts)
* **Backend:** Python (Flask), TextBlob (NLP), Newspaper3k (Scraping)
* **AI Engine:** Google Gemini 2.5 Flash API
* **Blockchain:** Custom Python-based Proof-of-Work Ledger
* **Search:** DuckDuckGo Search API (Agentic Retrieval)
